datacenter technology cluster

My team and i built a cluster of servers into a server rack in the datacenter, i focussed mainly on the security of the servers. I also helped with installing all the necessary software on the server like esxi and vcenter. I also configured part of the firewall

I liked this project alot i specifically when i installed wazuh a cybersecurity tool that scans our machines for safety and even scans files on the system with virustotal to check if they are not virusses i think it's super cool.

in the screenshot you can see the wazuh detecting a virus with virustotal. As you can see the first 3 scans returned 0 meaning it wasn't a virus but the other 2 returned 1 which means it is a virus. This happens automatically with all files that get added, deleted or changed in certain folders of your choice.
